react-howler-player

React package to render player UI with howler.js audio library.

Demo

Usage

Install the package
npm install --save react-howler-player
Import the package
import Player from "react-howler-player"
Pass an audio url in props.
import React, { Component, PureComponent } from 'react
import Player from "react-howler-player"

export default class App extends PureComponent {
  render() {
      return (
          <div>
            <Player
                src={[AUDIO_URL]}
            />
            <Player
                src={[AUDIO_URL]}   
                isDark={true}
            />
          </div>
      )
  }
}
